# Overview
Simulates a pizza ordering system using Tkinter for a graphical user interface. Features a variety of interface elements such as radio buttons, checkboxes and drop down menus as well as a dynamic tax calculation.

# How to Run
Download kevinTruongNokYinSoProjectFive.py and roofPizza.gif and place them in the save folder, then run kevinTruongNokYinSoProjectFive.py.
